Servicios para hacer webs

Un dominio es el nombre único que identifica a una web en internet y te permite visitarla tecleando en la barra de direcciones del navegador.

  • Namecheap
  • GoDaddy

El hosting es un servicio de alojamiento para páginas web bajo un servidor, que permite publicar un sitio web en Internet.

  • HostGator
  • Bluehost

Analytics es una herramienta de analítica web que proporciona información estadística detallada de un sitio web.

  • Google Analytics
  • Matomo
